# GAMEPORTAL 🎮

**GAMEPORTAL** es una plataforma web integral diseñada para entusiastas de los videojuegos. Ofrece una experiencia única que combina guías de optimización, una biblioteca completa de juegos para múltiples plataformas (PC, Xbox, PS3, Wii, DS) y herramientas técnicas esenciales para mejorar el rendimiento de hardware y software.

## 🚀 Demo en vivo
- **Sitio Web:** [gameportal-eloo.vercel.app](https://gameportal-eloo.vercel.app)
- **Repositorio:** [github.com/Troler2002/GAMEPORTAL](https://github.com/Troler2002/GAMEPORTAL)

## ✨ Características principales

- **Buscador Inteligente tipo Google**: Autocompletado dinámico categorizado por juegos y herramientas.
- **Centro de Descargas**: Sistema de descarga simulada con barra de progreso y feedback auditivo.
- **Modales Unificados**: Información detallada de cada título, incluyendo sinopsis, características y galería de personajes.
- **Muro de Comunidad**: Interacción en tiempo real donde los usuarios pueden publicar estados, dar "likes" y comentar.
- **Diseño Cyber-Minimalista**: Interfaz oscura con acentos neón rojos, optimizada para una estética gamer moderna.
- **Totalmente Responsivo**: Experiencia fluida tanto en computadoras de escritorio como en dispositivos móviles.
- **Secciones Especializadas**:
  - **PC/Consolas**: Catálogo detallado por plataforma.
  - **Tecnología**: Guías de mantenimiento y optimización (ej. AtlasOS, Acer Aspire 3).
  - **Video**: Integración multimedia para tutoriales y tráilers.
  - **Trucos**: Repositorio de códigos y secretos para juegos clásicos.

## ⚠️ Descargo de responsabilidad

Este proyecto ha sido desarrollado con fines exclusivamente educativos y de demostración técnica (Portfolio). **GAMEPORTAL no aloja, distribuye ni facilita la descarga de software ilegal o archivos protegidos por derechos de autor.** El sistema de "descarga" implementado es una simulación visual diseñada para mostrar habilidades de desarrollo frontend. Todas las marcas, nombres de juegos e imágenes son propiedad de sus respectivos dueños.

## 🛠️ Tecnologías utilizadas

- **HTML5**: Estructura semántica avanzada.
- **CSS3**: Variables CSS, Flexbox, Grid Layout y animaciones personalizadas.
- **JavaScript (ES6+)**: Lógica dinámica, delegación de eventos y manipulación del DOM sin dependencias externas.
- **Font Awesome**: Iconografía profesional.
- **Google Fonts**: Tipografía "Inter" para máxima legibilidad.

## 📦 Instalación local

1. Clona este repositorio:
   ```bash
   git clone https://github.com/Troler2002/GAMEPORTAL.git
   ```
2. Abre el archivo `index.html` en tu navegador preferido.
